Função RGBA

A função rgba() é usada para definir a cor de um elemento em uma página HTML, permitindo que você especifique a intensidade das cores vermelho, verde, azul e alpha (transparência).

Os valores de red, green e blue podem variar de 0 a 255, enquanto o valor de alpha varia de 0 a 1. Um valor de alpha de 0 significa que a cor é completamente transparente, enquanto um valor de 1 significa que a cor é completamente opaca.
