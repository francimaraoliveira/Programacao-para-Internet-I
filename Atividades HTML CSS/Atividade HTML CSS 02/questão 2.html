Gradiente de arco arco-íris

A referida questão foi feita utilizando o gradiente linear para a direita.

Este código CSS criará um gradiente horizontal que começa com a cor vermelha no lado esquerdo e progride para o violeta no lado direito. A propriedade linear-gradient especifica o tipo de gradiente que queremos criar, enquanto to right indica a direção do gradiente.
